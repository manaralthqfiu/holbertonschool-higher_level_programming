The ``5-text_indentation`` module
=================================

Using ``text_indentation``
--------------------------
    >>> text_indentation = __import__('5-text_indentation').text_indentation

Normal Case:
    >>> text_indentation("Hello. How are you? Fine: thanks.")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    Fine:
    <BLANKLINE>
    thanks.
    <BLANKLINE>

Checking for leading/trailing spaces:
    >>> text_indentation("  Wait.   What? ")
    Wait.
    <BLANKLINE>
    What?
    <BLANKLINE>
